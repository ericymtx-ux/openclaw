# 智能投研产品报表设计方案

**创建日期**: 2026-02-04
**版本**: v1.0

---

## 核心定位

**辅助决策工具**，不是"跟投系统"。

AI 提供：
- 候选清单 (What)
- 置信度 (How sure)
- 风险提示 (Watch out)

人类负责：
- 最终决策 (Yes/No)
- 仓位管理
- 执行下单

## 一、评测机制设计

### 1.1 核心指标体系

```
┌─────────────────────────────────────────────────────────────┐
│                    信号有效性评估体系                         │
├─────────────────────────────────────────────────────────────┤
│  维度 1: 基础统计                                            │
│  ├── 命中率 (Hit Rate)                                      │
│  │   = 盈利信号数 / 总信号数 × 100%                          │
│  │   衡量: 信号准确吗？                                       │
│  │                                                          │
│  ├── 盈亏比 (Risk/Reward)                                   │
│  │   = 平均盈利 / 平均亏损                                    │
│  │   衡量: 赚多亏少吗？                                       │
│  │                                                          │
│  ├── 期望值 (Expected Value)                                │
│  │   = 命中率 × 平均盈利 - (1-命中率) × 平均亏损              │
│  │   衡量: 长期能赚钱吗？                                     │
│  │                                                          │
│  └── 夏普比率 (Sharpe Ratio)                                │
│      = (平均收益 - 无风险利率) / 收益标准差                   │
│      衡量: 风险调整后的收益                                   │
├─────────────────────────────────────────────────────────────┤
│  维度 2: 分类统计                                            │
│  ├── 按信号类型                                              │
│  │   ├── 🔴开 (买入信号) 单独统计                             │
│  │   ├── 🟡持 (持有信号) 单独统计                             │
│  │   └── ⚪空 (观望信号) 单独统计                             │
│  │                                                          │
│  ├── 按行业板块                                              │
│  │   ├── 有色金属                                            │
│  │   ├── 电力设备                                            │
│  │   └── ...                                                 │
│  │                                                          │
│  └── 按时间段                                                │
│      ├── 月度 (当月 vs 历史平均)                             │
│      ├── 周度 (本周信号表现)                                 │
│      └── 季度 (季度胜率趋势)                                 │
├─────────────────────────────────────────────────────────────┤
│  维度 3: 动态评估                                            │
│  ├── 信号置信度                                              │
│  │   = 热度排名 × 0.3 + 板块联动 × 0.3 + 历史胜率 × 0.4     │
│  │                                                          │
│  ├── 市场环境                                                │
│  │   ├── 上涨市: 命中率基准 60%                              │
│  │   ├── 震荡市: 命中率基准 50%                              │
│  │   └── 下跌市: 命中率基准 40%                              │
│  │                                                          │
│  └── 信号新鲜度                                              │
│      └── 最新信号 > 3日前信号 (信号会过时)                    │
└─────────────────────────────────────────────────────────────┘
```

```
验证策略: 多窗口验证，避免短期噪音

├── 1日验证 (次日走势)

│   指标: 次日涨跌幅
│
├── 3日验证 (短期趋势)

│
├── 5日验证 (一周表现)
│   目的: 周度表现
│   指标: 5日累计涨跌幅
│
└── 10日验证 (中期表现)
    目的: 中期趋势
    指标: 10日累计涨跌幅
```

## 四、关键产品决策

### 4.1 置信度计算公式

```
置信度 = (热度排名得分) × 0.3 + (板块联动得分) × 0.3 + (历史胜率) × 0.4

其中:
- 热度排名得分 = 1 / log2(排名 + 2)  # 排名越高得分越高
- 板块联动得分 = 1 + (同板块信号数 × 0.2)  # 联动越多得分越高
- 历史胜率 = 该类信号历史命中率  # 历史表现越好得分越高
```

### 4.2 风险等级划分

| 等级 | 标签 | 条件 | 建议 |
|------|------|------|------|
| 🟢 低风险 | 正常 | 情绪指数 0.5-1.5 | 正常操作 |
| 🟡 中风险 | 关注 | 情绪指数 > 1.5 或 < 0.5 | 谨慎操作 |
| 🟠 高风险 | 警示 | 单一板块 > 3 只同向信号 | 轻仓 |
| 🔴 极高风险 | 警告 | 证伪信号出现 | 观望 |

### 4.3 信号时效性

```
信号有效期:
├── 🔴买入信号: 3 个交易日
├── 🟡持有信号: 5 个交易日
└── ⚪观望信号: 7 个交易日

过期处理:
└── 自动移至历史记录，标记"已过期"
```

## 六、总结

### 报表设计的核心原则

1. **分层展现**: 1 分钟快览 → 5 分钟详情 → 30 分钟深度
2. **置信度量化**: 让用户知道"多确定"
3. **风险可视化**: 明确提示"要注意什么"
4. **历史可追溯**: 胜率统计 → 信任建立
5. **反馈闭环**: 表现追踪 → 持续优化

### 差异化价值

| 维度 | 普通工具 | 本系统 |
|------|----------|--------|
| 信号来源 | 单一 | 邮件 + 人工 |
| 置信度 | 无 | ⭐⭐⭐⭐ |
| 风险提示 | 无 | ⚠️ 明确 |
| 历史表现 | 无 | 📈 可追溯 |
| 决策辅助 | 直接推票 | 辅助决策 |
