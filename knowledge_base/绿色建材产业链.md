# 绿色建材产业链调研

> 调研时间: 2026-02-03  
> 政策驱动: 一号文"绿色建材下乡"

---

## 一、产业链全景

```
原材料 ──→ 加工 ──→ 制品 ──→ 应用
  │         │         │         │
 水泥      保温材料   装配式    建筑
 石英砂    防水材料   绿色涂料   装修
 纤维      门窗       BIPV     家居
```

## 二、核心环节

### 2.1 保温材料

| 品类 | 代表企业 |
|------|----------|
| 岩棉 | 鲁阳节能 |
| 聚氨酯 | 东方雨虹 |

### 2.2 防水材料

| 企业 | 地位 |
|------|------|

| 科顺股份 | 防水第二 |
| 北新建材 | 防水新锐 |

### 2.3 装配式建筑

| 品类 | 代表企业 |
|------|----------|
| PC构件 | 远大住工、筑友制造 |
| 钢结构 | 鸿路钢构、精工钢构 |
| 加气板 | 北新建材(金隅) |

### 2.4 绿色涂料

| 企业 | 特点 |
|------|------|
| 三棵树 | 墙面漆龙头 |
