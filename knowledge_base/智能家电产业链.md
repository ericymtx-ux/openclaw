# 智能家电产业链调研

> 调研时间: 2026-02-03

---

## 一、产业链全景

```
上游(零部件) ──→ 中游(整机) ──→ 下游(渠道)
     │              │              │
  压缩机          白电(空冰洗)    线上(京东/天猫)
  电机            厨电(烟灶消)    线下(苏宁/专卖店)
  控制器          小家电          工程(精装房)
  零部件          智能家居
```

---

## 二、产业链详解

### 2.1 白电 (空调/冰箱/洗衣机)

| 环节 | 代表企业 | 特点 |
|------|----------|------|
| 空调 | 美的、格力、海尔 | 三足鼎立 |
| 冰箱 | 海尔、美的、容声 | 海尔高端领先 |
| 洗衣机 | 海尔、美的、小天鹅 | 海尔绝对龙头 |

**核心零部件**:

| 零部件 | 龙头 | 逻辑 |
|--------|------|------|
| 压缩机 | 美的(美芝)、格力(凌达) | 自供为主 |
| 电机 | 东方电热 | 家电PTC龙头 |
| 阀件 | 三花智控 | 全球阀件龙头 |
| 热管理 | 盾安环境 | 格力系 |

### 2.2 厨电

| 品类 | 龙头 | 特点 |
|------|------|------|
| 油烟机 | 老板电器、方太 | 老板线下领先 |
| 集成灶 | 火星人、亿田 | 集成灶渗透率提升 |
| 洗碗机 | 西门子、美的 | 洗碗机普及中 |

### 2.3 小家电

| 品类 | 龙头 | 逻辑 |
|------|------|------|
| 扫地机 | 科沃斯、石头科技 | 渗透率提升 |
| 洗地机 | 添可(科沃斯)、追觅 | 高速增长 |
| 厨房小家电 | 苏泊尔、九阳、美的 | 成熟稳定 |

### 2.4 智能家居

| 品类 | 龙头 |
|------|------|
| 智能照明 | 欧普照明、雷士 |
| 智能安防 | 海康威视、大华 |
| 全屋智能 | 华为、小米、美的 |

## 三、选股逻辑

### 白电 (成熟稳健)

| 标的 | 逻辑 |
|------|------|
| 美的集团 | 综合家电龙头，全球化 |
| 格力电器 | 空调龙头，高股息 |
| 海尔智家 | 高端化，卡萨帝领先 |
| 海信家电 | 央空+冰箱，受益地产 |

### 零部件 (成长性)

| 标的 | 逻辑 |
|------|------|
| 三花智控 | 热管理龙头，车+家双轮 |
| 东方电热 | 家电PTC+新能源车PTC |
| 盾安环境 | 格力系，热管理 |

### 小家电 (分化)

| 标的 | 逻辑 |
|------|------|
| 科沃斯 | 扫地机龙头 |
| 石头科技 | 扫地机新锐 |
| 苏泊尔 | 小家电稳健 |
