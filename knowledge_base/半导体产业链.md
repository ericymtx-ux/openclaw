# 半导体产业链深度调研

> 调研时间: 2026-02-03  
> 数据来源: 公开市场研究 + 产业链分析

## 一、产业链全景图

```
设计 (Fabless) ──→ 制造 (Foundry) ──→ 封测 (OSAT) ──→ 应用
     │                  │                   │              │
  EDA工具           晶圆代工            封装测试         消费电子
  IP核             设备                材料            汽车电子
  IC设计           材料                分立器件         工业/AI
                       │
                       └──→ 设备: 光刻/刻蚀/薄膜/检测
                           └──→ 材料: 硅片/光刻胶/特气/靶材
```

## 三、价格/周期传导

```
半导体周期: 4-5年一轮
  复苏 → 繁荣 → 衰退 → 萧条

当前 (2024-2025): 复苏期
  - 存储率先反弹
  - 消费电子缓慢复苏
  - AI芯片需求旺盛
```

**核心价格指标**:
1. 存储芯片价格 (DRAM/NAND)
2. 硅片价格 (12寸/8寸)
3. 设备订单 (北美/日本半导体设备)

## 五、选股逻辑

| **北方华创** | 平台型龙头，覆盖80%设备 |

| **拓荆科技** | 薄膜设备，国产第一 |
| **盛美上海** | 清洗设备龙头 |
| **中科飞测** | 检测设备新锐 |

**买入时机**:
- 晶圆厂招标 → 设备订单增加
- 国产替代加速

## 六、风险提示

1. **设备禁运**: 美国/荷兰设备限制
2. **需求不及预期**: 消费电子疲软
3. **产能过剩**: 成熟制程扩产太快
4. **技术差距**: 先进制程追赶慢

## 八、结论

| 环节 | 投资建议 | 核心逻辑 |
|------|----------|----------|
| 设备 | ✅ 成长 | 国产替代最确定 |
| 材料 | ✅ 成长 | 替代空间大 |
| 设计 | ⚠️ 分化 | AI芯片强，传统弱 |
| 封测 | ✅ 优选 | 先进封装需求旺 |
| 制造 | ⚠️ 观望 | 周期底部 |

**当前策略**:
- **首选**: 北方华创/中微公司 (设备龙头)
- **次选**: 长电科技/通富微电 (先进封装)
- **关注**: 寒武纪 (AI芯片)
- **回避**: 成熟制程制造 (产能过剩)
