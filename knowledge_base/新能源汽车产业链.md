## 一、产业链全景图

```
上游(原材料/零部件)          中游(整车制造)          下游(服务)
     │                            │                    │
  锂矿/钴矿              ────→   纯电/插混/氢燃料   ────→  充电桩/换电站
  正极材料                        │                    │
  负极材料                   比亚迪/特斯拉            维修保养
  电解液                       /  \                  汽车租赁
  隔膜                      上汽/理想                 二手车
  锂电设备                    │
  电池壳                      └─→ 智能化(智驾/座舱)
                             
  驱动电机              核心三电: 电池/电机/电控
  电控系统              智能驾驶: 域控/激光雷达/摄像头
```

## 二、产业链详解

### 2.1 上游 - 关键原材料

| 环节 | 核心材料 | 价格指标 | 代表企业 |
|------|----------|----------|----------|
| 锂矿 | 碳酸锂/氢氧化锂 | 碳酸锂价格 (万元/吨) | 赣锋锂业、天齐锂业、盐湖股份 |
| 钴矿 | 硫酸钴/四氧化三钴 | 钴价 (万元/吨) | 华友钴业、寒锐钴业 |
| 镍矿 | 硫酸镍 | 镍价 (万元/吨) | 格林美、中伟股份 |
| 石墨 | 负极材料 (人造/天然) | 石墨价格 | 贝特瑞、璞泰来 |
| 隔膜 | 湿法/干法隔膜 | 隔膜价格 (元/㎡) | 恩捷股份、星源材质 |
| 电解液 | 六氟磷酸锂/LiFSI | 六氟价格 (万元/吨) | 天赐材料、新宙邦 |

**关键价格跟踪**:
- 碳酸锂: 主力跟踪指标，周期波动大
- 六氟磷酸锂: 电解液成本核心
- 负极石墨: 相对稳定

### 2.2 上游 - 核心零部件

**动力电池** (最核心环节，占整车成本 30-40%)

| 电池类型 | 特点 | 代表企业 |
|----------|------|----------|
| 磷酸铁锂 | 成本低、安全性好、能量密度稍低 | 宁德时代、比亚迪、国轩高科 |
| 三元锂电池 | 能量密度高、成本高、安全性稍弱 | 宁德时代、孚能科技、中创新航 |
| 固态电池 | 下一代技术、能量密度翻倍 | 卫蓝新能源、赣锋锂业(布局) |

**电驱系统** (决定加速/能耗)

| 环节 | 代表企业 |
|------|----------|
| 电机 | 方正电机、汇川技术、英搏尔 |
| 电控 | 汇川技术、精进电动 |
| 减速器 | 青山工业、双环传动 |

## 三、价格传导机制

```
原材料上涨 → 电池成本上升 → 整车提价/降本 → 销量变化
                                              ↓
                                       碳酸锂↓ → 电池↓ → 整车降价促销
```

**核心价格敏感指标**:
1. **碳酸锂价格**: 电池成本核心，领先指标
2. **锂电材料价格**: 六氟/电解液/隔膜
3. **芯片价格**: 智能化成本

**传导时滞**: 原材料 → 电池 (1-2月) → 整车 (2-3月)

## 五、选股逻辑

### 5.1 上游周期股 (碳酸锂相关)

**核心逻辑**: 碳酸锂价格周期波动

| 标的 | 逻辑 |
|------|------|
| 赣锋锂业 | 锂资源全布局，弹性大 |
| 天齐锂业 | 澳洲矿成本低 |
| 盐湖股份 | 盐湖提锂，成本优势 |

**买卖时机**:
- 碳酸锂跌至 5万/吨以下 → 布局
- 碳酸锂涨至 20万/吨以上 → 兑现

### 5.2 中游成长股 (电池/材料)

**核心逻辑**: 渗透率提升 + 技术迭代

| 标的 | 逻辑 |
|------|------|
| 宁德时代 | 电池龙头，估值锚 |
| 比亚迪 | 整车+电池一体化 |
| 恩捷股份 | 隔膜龙头 |
| 天赐材料 | 电解液龙头 |
| 汇川技术 | 电控龙头 |

**买入时机**:
- 碳酸锂价格企稳回升
- 整车销量超预期
- 技术突破 (固态电池/800V)

### 5.3 整车股

**核心逻辑**: 销量份额 + 智能化溢价

| 标的 | 逻辑 |
|------|------|
| 比亚迪 | 全价格带覆盖 |
| 理想汽车 | 增程细分龙头 |
| 小鹏汽车 | 智驾技术领先 |
| 赛力斯 | 华为合作 |

### 5.4 智能化

**核心逻辑**: L3 落地 + 渗透率提升

| 标的 | 逻辑 |
|------|------|
| 德赛西威 | 域控制器龙头 |
| 禾赛科技 | 激光雷达龙头 |
| 四维图新 | 高精地图 |
| 地平线 | 智驾芯片 |

## 六、风险提示

1. **碳酸锂价格波动**: 周期性强，买卖节奏难把握
2. **整车价格战**: 毛利率承压
3. **智能化不及预期**: 技术路线变化
4. **政策退坡**: 补贴减少

## 八、结论

| 环节 | 投资建议 | 核心逻辑 |
|------|----------|----------|
| 锂矿 | ⚠️ 周期股 | 碳酸锂价格周期 |
| 电池材料 | ✅ 成长股 | 龙头集中度提升 |
| 电池 | ✅ 成长股 | 宁德/比亚迪双寡头 |
| 整车 | ⚠️ 分化 | 看份额+智能化 |
| 智能化 | ✅ 成长股 | L3 落地元年 |

**当前策略**:
- 碳酸锂低位 → 布局上游锂矿
- 整车价格战 → 关注龙头比亚迪
- 智能化加速 → 布局德赛西威/禾赛
