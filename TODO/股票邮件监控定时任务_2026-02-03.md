# 股票邮件监控定时任务

**创建日期**: 2026-02-03
**状态**: ✅ 已完成
**优先级**: P1

---

## 需求

- 每半小时检查邮件（工作日 8:00-18:00）
- 筛选股票/ETF 相关邮件
- 提取【🔴开】【🟡持】【⚪空】信号
- **直接发送完整报告**（含量价关系）

---

## 输出格式

### 买入信号
```
## 🟢 **湖南黄金 (002155.SZ)** 【🟢 买入】
- 💰 收盘价: 35.05 | 📈 涨跌幅: +5.26%
- 📊 量价关系: 量缩价涨
```

### 持有信号
```
## ⭐ **利欧股份 (002131.SZ)** 【🔵 持有】
- 💰 收盘价: 9.89 | 📈 涨跌幅: -0.20%
- 📊 量价关系: 量缩价平
```

### 平仓信号
```
- 蓝色光标 (300058.SZ) 【🟡 平仓】
  💰 21.41 | 📈 +0.19% | 📊 量缩价平
```
