## 技术要点

### fast-browser-use
- 这是一个 Rust 版本的 browser-use 库
- 通过 MCP 协议控制浏览器
- 比 Puppeteer 快 10 倍以上，内存占用 15MB vs 100MB+
- 安装方式：`cargo add browser-use` 然后 `cargo run --bin mcp-server`
- SKILL 文件已同步到三个平台

### QuantSeek 框架
- 基于 Python 的极简量化投研框架
- 覆盖大A，可拓展到商品、BI
- 目前暂无开源仓库，可能是内部使用
- 定位：新手都能看懂但效率极高

## AI 编程工具使用指南 (新增)

**位置**: `skills/coding-agent/AI编程工具使用指南_2026-02-01.md`

### 工具选择决策树

```
任务
  │
  ├── 复杂度高？ ──是──→ Claude Code (opus)
  │       │
  │       └── 否──→ 需要并行多个？ ──是──→ Claude Team / OpenCode Team
  │                   │
  │                   └── 否──→ 纯 MCP 调用？ ──是──→ mcporter
  │                               │
  │                               └── 否──→ 小修改？ ──是──→ OpenCode
```

### 核心规则

| 工具 | 使用场景 |
|------|----------|
| **Claude Code** | 新项目初始化、架构设计、复杂调试 |
| **OpenCode** | Bug 修复、小功能迭代、代码优化 |
| **Claude Team** | 并行开发、批量处理、Beads 集成 |
| **OpenCode Team** | 简单并行任务 |
| **mcporter** | MCP Tool 直接调用 |

### 重要注意事项

1. **Claude Code 禁止在以下目录运行**:
   - `~/clawd/` (OpenClaw 主目录)
   - `~/Projects/openclaw/` (Live OpenClaw)

2. **Coding Agent 必须使用 PTY**:
   ```bash
   # ✅ 正确
   bash pty:true command:"codex exec 'task'"
   
   # ❌ 错误
   bash command:"codex exec 'task'"
   ```

3. **Claude Team Worker 需要**:
   - `skip_permissions: true`
   - `use_worktree: true` (推荐)

### 命令速查

```bash
# Claude Code
claude "task"              # 交互模式
claude -p "task"           # 头模式
claude --model opus        # 复杂任务

# OpenCode
opencode run "task"

# Claude Team (通过 mcporter)
mcporter call claude-team.spawn_workers workers='[{"project_path":"auto","bead":"cp-123"}]'
mcporter call claude-team.list_workers
mcporter call claude-team.close_workers session_ids='["Groucho"]'

# mcporter
mcporter list                          # 列出 server
mcporter call server.tool key=value    # 调用工具
```
